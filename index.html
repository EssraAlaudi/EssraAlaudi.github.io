Enhancement Two: Algorithms and Data Structures

I chose to include this artifact in my ePortfolio because it aligns well with the Algorithms and Data Structures category and offered opportunities for improvement. While the course provided much of the foundational code, the task was to complete and enhance the deep Q-learning implementation.This type of assignment showcases the ability to work with real machine-learning techniques and demonstrates an understanding of how algorithms guide intelligent decision-making.

the project was enhanced by refining the priority calculation through squaring the temporal-difference (TD) error. This emphasizes larger mistakes, helping the agent focus on more useful experiences. Additionally, a second enhancement was made by adding priority decay, gradually lowering the importance of past experiences unless they are re-sampled and updated. This keeps the agent’s attention on more recent and relevant situations. Together, these enhancements make the learning process more efficient and effective, ultimately improving the agent’s ability to generalize.

Course outcomes met: #3 and #4.

These outcomes were demonstrated through these enhancements by applying algorithmic principles to refine the agent’s learning behavior. Design choices like squaring the TD error and adding priority decay reflect an understanding of trade-offs. Additionally, the solution was extended beyond its original state using well-founded and innovative techniques that add value. .

The process of modifying and updating the treasure hunt game deepened my understanding of reinforcement learning, particularly how different computing techniques can influence an agent’s learning behavior. I also learned how small changes can make a big difference - like how squaring the TD error or introducing priority decay significantly impacts the quality and efficiency of the agent’s learning. That being said the process did come with a challenge because upon getting started, I found out my original enhancement plan was already implemented. While unexpected, this shift pushed me to search for other meaningful ways the project could be enhanced, while still staying close to the original plan.
