Enhancement One: Software Design and Engineering

The reason I chose to include this inventory app in my ePortfolio is because it fits into the category of software engineering and design and showcases several important skills in software development. These skills include working with a database, handling permissions and error conditions, designing a user-friendly interface, and applying role-based access control. The artifact was improved through enhancements such as adding a search feature, implementing role-based restrictions, and refining both the layout and documentation.

Course outcomes met: 3, 4, and 5.

These outcomes were demonstrated through this project because it was designed using practical programming structures and logic. Features like search functionality, role-based access, and database interactions reflect the application of algorithmic principles while utilizing well-founded techniques, skills, and tools in computing practices. Additionally, protective design principles were incorporated to anticipate potential vulnerabilities and promote privacy and security, such as limiting access through roles and using invite codes for controlled user registration.

The process of enhancing and modifying the artifact involved refinement, testing, and continuous adjustments. I also visited a variety of resources during the development of the enhancements. Through this experience, I became more aware of how much I still have to learn and feel increasingly motivated to grow.

Some specific challenges I faced during this enhancement included trying to figure out how to implement the search functionality, something I wasn't able to complete the last time I worked on this project, and better understanding how the role-based access control would work. Additionally, I also found it difficult to design and organize the new layout for the Admin inventory screen in a way that made sense. Although I needed help along the way, this project helped me recognize the value of seeking guidance, learning by doing, and staying persistent even when things feel difficult.
